### 每模块新增题型 (在现有基础上+10题/模块)

新增三类深度题，每模块至少+3物理原理题 +3工艺变化题 +4综合应用题：

#### 题型A: 物理原理题 (Physics Principles)
考察对底层物理机制的理解，而非记忆事实。

**示例:**
```json
{
  "id": "L13-PHOTO-PHYS-001",
  "module_code": "13-photolithography",
  "difficulty": "L3",
  "qtype": "single",
  "question": "EUV光刻中，为什么需要使用反射式光学系统而非传统的透射式？",
  "options": [
    "A. 几乎所有材料在13.5nm波长下都有强烈吸收，无法制作透明透镜",
    "B. 反射镜成本更低",
    "C. 反射系统体积更小",
    "D. 透射系统无法校正像差"
  ],
  "answer": "A",
  "explanation": "EUV 13.5nm光子能量约92eV，远大于材料带隙，几乎所有材料（包括空气）对该波长都有强烈吸收。因此必须用多层Mo/Si布拉格反射镜（约70%反射率）构成全反射光路，整个系统需在真空中工作。"
}
```

**需覆盖的物理原理主题 (每模块):**
- 光刻: 衍射极限 (Rayleigh criterion k1*λ/NA)、驻波效应、光子散粒噪声
- 薄膜: 台阶覆盖模型、应力起源 (本征/热失配)、成核理论
- 刻蚀: 离子轰击物理溅射产额、侧壁钝化化学、ARDE/微负载效应
- 外延: 晶格匹配临界厚度、Stranski-Krastanov生长模式
- CMP: Preston方程、接触力学模型
- 注入: 离子射程分布(LSS理论)、沟道效应、损伤退火动力学
- 计量: SEM二次电子产额、椭偏仪Δ/Ψ参数物理含义
- DTCO: 短沟道效应(DIBL/SS退化)、RC延迟模型

#### 题型B: 工艺变化题 (Process Variation)
考察参数变化对器件/良率的影响。

**示例:**
```json
{
  "id": "L13-PHOTO-VAR-001",
  "module_code": "13-photolithography",
  "difficulty": "L3",
  "qtype": "single",
  "question": "在28nm节点，如果光刻胶的PEB温度偏离设定值±2°C，最可能导致的后果是？",
  "options": [
    "A. CD均匀性恶化，化学放大反应速率改变导致线宽偏移",
    "B. 光刻胶完全失效无法显影",
    "C. 仅影响显影速度不影响CD",
    "D. 只对EUV有影响，对DUV无影响"
  ],
  "answer": "A",
  "explanation": "化学放大光刻胶(CAR)的PEB过程是酸催化扩散反应，对温度极其敏感。±2°C的温度波动会导致酸扩散长度变化，进而改变最终CD（典型灵敏度~5-10nm/°C），是先进节点CD均匀性控制的关键因素之一。"
}
```

**需覆盖的工艺变化主题:**
- CD均匀性 (线宽粗糙度LWR/LER来源)
- 膜厚均匀性 (沉积/刻蚀速率漂移)
- 掺杂浓度偏差 (注入剂量/能量波动)
- 对准/套刻误差 (overlay budget分解)
- 缺陷密度与良率模型 (Poisson/Bose-Einstein/负二项)
- 工艺窗口 (Focus-Exposure Matrix, DOF)
- 热预算漂移对器件特性的影响

#### 题型C: 综合应用题 (Application)
跨模块关联，贴近实际PIE工作。

**示例:**
```json
{
  "id": "L13-PHOTO-APP-001",
  "module_code": "13-photolithography",
  "difficulty": "L3",
  "qtype": "single",
  "question": "某批次晶圆出现光刻胶图形桥接(bridging)缺陷，且集中在晶圆边缘。最合理的排查方向是？",
  "options": [

    "B. 直接更换光刻胶供应商",
    "C. 降低曝光剂量对整片晶圆统一补偿",
    "D. 延长PEB时间"
  ],
  "answer": "A",
  "explanation": "边缘桥接缺陷通常与光刻胶膜厚均匀性差有关——边缘膜厚偏厚导致曝光不足，显影不净形成桥接。应检查旋涂曲线、EBR(Edge Bead Removal)设定、以及边缘区域的膜厚和CD数据。统一补偿曝光剂量可能造成中心区域过曝。"
}
```

## P1: 互动可视化模块

在 `streamlit_app.py` 中新增 "📐 可视化实验室" tab，包含以下交互式模块。

### 模块1: 光刻衍射极限模拟器 (Lithography Diffraction Simulator)
- 三个滑块: NA (0.3-1.4), 波长λ (13.5/193/248/365nm), k1因子 (0.25-0.8)
- 实时计算并显示最小半周期分辨率: R = k1 * λ / NA
- Matplotlib绘制: 线宽 vs NA/λ 的二维热力图
- 标注实际工艺节点位置 (28nm/14nm/7nm/3nm等)

### 模块2: CMP Preston方程计算器
- 滑块: 压力P、相对速度V、Preston系数Kp
- 实时显示: 移除速率 RR = Kp * P * V
- 附加: 图案密度对局部速率的影响曲线

### 模块3: 薄膜光学常数 (n, k) 浏览器
- 下拉选择材料: Si, SiO2, Si3N4, TiN, Cu, Al, photoresist
- 波长范围滑块
- 绘制 n(λ) 和 k(λ) 双Y轴曲线
- 显示选定波长处的n, k值及对应反射率 R = ((n-1)²+k²)/((n+1)²+k²)

### 模块4: 掺杂浓度-电阻率换算器
- 滑块: 掺杂浓度 (1e14 - 1e20 cm⁻³)
- 选择掺杂类型: N型(P/As/Sb)、P型(B)
- 实时计算并显示电阻率 (基于Irvin曲线数据)
- 显示对应的方块电阻 R_sheet = ρ/t (t为结深)

### 模块5: 互连RC延迟模型
- 输入: 线宽W、线间距S、线高H、介电常数k、金属电阻率ρ
- 实时计算: R(每单位长度)、C(每单位长度)、RC延迟
- 对比: 不同材料 (Cu/low-k vs Al/SiO2) 的RC延迟柱状图

### 技术实现
- 使用 `st.slider`、`st.selectbox` 做交互控件
- 使用 `matplotlib` (Agg后端) 绘图，`st.pyplot` 显示
- 每个模块封装为独立函数 `def viz_lithography()` `def viz_cmp()` 等
- 在 "📐 可视化实验室" tab 中用 `st.expander` 折叠每个模块
